
RexxIPC Source Distribution.
------- ------ -------------

This is a release of the source files for RexxIPC.

As the author (Serge Brisson) has not kept an OS/2
environment, these source files are released as they
were for the last binary release (REXXIPC V1.30-001)
in 1997.

The file 'LICENSE.txt' contains informations on the
terms of distribution.

These files are released in the hope that they may
be useful or instructive for someone.
